# 💰 Personal Finance Tracker

A full-stack web application to track income/expenses, manage budgets, and achieve savings goals.

## 🌐 Live Demo
- Frontend:(https://finance-tracker-zeta-three-24.vercel.app/)
- Backend: (https://finance-tracker-backend-nwon.onrender.com)

## 🛠️ Tech Stack
- **Frontend:** React, Recharts
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Auth:** JWT + Bcrypt
- **Deployment:** Vercel (Frontend) + Render (Backend)

## ✨ Features
- 🔐 User Authentication (Register/Login)
- 💸 Add/Delete Income & Expense Transactions
- 📊 Visual Dashboard (Pie Chart + Bar Chart)
- 🎯 Budget Manager with warnings
- 🏦 Savings Goals tracker
- 🔑 Forgot Password via Email
